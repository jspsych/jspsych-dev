Add `@jspsych/new-multiplayer-adapter`, a CLI that scaffolds a new jsPsych multiplayer adapter package. The generated package ships a TypeScript stub implementing the `MultiplayerAdapter` interface plus the standard build/test/docs/example/citation tooling. When run inside the jspsych-multiplayer repository it creates the package under `./packages` and names it under the `@jspsych-multiplayer` npm scope.
